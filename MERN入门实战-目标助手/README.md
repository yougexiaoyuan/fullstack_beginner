## 项目简介
通过目标驱动自我的任务，高效的帮助个人用户完成每天的工作和生活任务。

### 功能描述
大概2个页面功能，一个目标列表页面，一个添加目标的明细页面。

### 代码架构
采用Node.js后端，和React前端技术框架，非常简单的框架和代码适合入门学习者。

### 代码评审

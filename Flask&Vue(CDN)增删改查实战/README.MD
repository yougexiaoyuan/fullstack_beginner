1 创建项目文件夹，创建Flask 主文件app.py，并粘贴案例代码

2 安装Flask跨域扩展 Flask-CORS extension https://flask-cors.readthedocs.io/
$ pip install -U flask-cors

3 测试app.py，flask run 命令，运行后端的web server。

4  安装集成工具phpstudy，启动mysql数据库

5 安装mysql 客户端工具；并且创建数据库testingdb，执行生成的数据表sql

6 测试app.py 是否正常联通数据库

7 创建demo.html静态文件，并复制案例代码

8 浏览器打开demo.html代码测试。
